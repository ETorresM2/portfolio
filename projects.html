React Messaging App

Messaging browser app using React for the client, users can create accounts and add other users as contacts. They can send and receive messages in real time. Account security is ensured via password encryption using Bcrypt, and endpoints are protected using sessions, CORS is used to add a further level of security to the app. Real time messsaging is handled through Pusher, an API that is used in the front and back end to allow real-time communication by utilizing web sockets.
